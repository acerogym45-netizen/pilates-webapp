## 📊 기술 세부사항

### KST 날짜 체크 로직
```javascript
// UTC → KST 변환
const kstOffset = 9 * 60 * 60 * 1000;
const kstDate = new Date(now.getTime() + kstOffset);
const currentDay = kstDate.getUTCDate(); // KST 날짜

// 자동 승인 기간 체크
const isAutoApprovalPeriod = currentDay >= 3 && currentDay <= 10;
```

### 상태 분기
```javascript
const status = isAutoApprovalPeriod ? 'approved' : '접수';
const adminNote = isAutoApprovalPeriod 
    ? `자동 승인 (${currentMonth}월 ${currentDay}일 접수)` 
    : `관리자 검토 필요 (${currentMonth}월 ${currentDay}일 접수)`;
```

## 💬 FAQ

**Q1. 왜 3~10일로 변경했나요?**  
A1. 사용자 편의성 향상을 위해 상시 접수를 허용하되, 특정 기간에만 자동 승인하여 관리자 부담을 줄이기 위함입니다.

**Q2. 기간 외 접수된 해지 신청은 어떻게 처리하나요?**  
A2. 관리자 페이지 → 해지 관리 탭에서 확인 후 수동으로 상태를 변경할 수 있습니다.

**Q3. 기존 데이터에 영향이 있나요?**  
A3. 없습니다. 기존 해지 신청 데이터는 그대로 유지됩니다.

**Q4. 자동 승인 기간을 변경하고 싶다면?**  
A4. `js/main.js` 파일에서 다음 부분을 수정하세요:
```javascript
const isAutoApprovalPeriod = currentDay >= 3 && currentDay <= 10;
// 예: 5~15일로 변경하려면
// const isAutoApprovalPeriod = currentDay >= 5 && currentDay <= 15;
```
