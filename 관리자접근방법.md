# 🔐 관리자 페이지 접근 방법

## 🎯 숨겨진 트리거

입주민 페이지에서 **"청주SK뷰자이"** 로고를 **2초 안에 5번 빠르게 클릭**하세요!

### 📱 모바일에서
1. 입주민 페이지 접속
2. 상단 "청주SK뷰자이" 텍스트를 손가락으로 톡톡톡톡톡 (5번)
3. 관리자 페이지로 자동 이동

### 💻 PC에서
1. 입주민 페이지 접속
2. 상단 "청주SK뷰자이" 텍스트를 마우스로 빠르게 5번 클릭
3. 관리자 페이지로 자동 이동

- 트리거 방식은 긴급 상황용
