# 청주SK뷰자이 필라테스 관리자 페이지

마스터플랜리소스 × 청주SK뷰자이 필라테스 프로그램 관리자 전용 사이트입니다.

## 🎯 프로젝트 개요

이 사이트는 입주민 신청 사이트와 **완전히 분리된 별도 도메인**으로 운영됩니다.
관리자만 접근 가능하며, 입주민들은 이 사이트의 존재를 알 수 없습니다.

## 🔗 시스템 구조

```
[입주민 사이트]                      [관리자 사이트]
도메인: 청주sk뷰자이-필라테스.com    도메인: 관리자-전용-도메인.com
- 레슨 신청                          - 신청 관리
- 문의 작성                          - 문의 관리
- 공개 문의 조회                     - 승인/거부 처리
                                     - CSV 다운로드
         ↓                                  ↓
              [공유 데이터베이스]
              - pilates_contracts
              - pilates_inquiries
```

## ✨ 주요 기능

### 📋 신청 관리 (index.html)
- 대시보드 통계 (대기중/승인/거부)
- 신청 목록 및 상세 조회
- 서명 이미지 확인
- 승인/거부 처리
- CSV 다운로드
- 실시간 검색 및 필터링

### 💬 문의 관리 (inquiry.html)
- 문의 통계 (대기중/완료/공개/전체)
- 공개/비공개 문의 모두 조회
- 답변 작성 및 전송
- 상태 자동 업데이트

## 🚀 배포 가이드

### 전체 프로젝트 한 번에 배포
1. 루트 프로젝트를 Publish 탭에서 배포
2. 배포된 도메인 확인 (예: https://skview-pilates.com)

### 접근 경로
```
입주민 페이지: https://skview-pilates.com/
관리자 페이지: https://skview-pilates.com/admin/
```

### 설정 완료
- ✅ API 경로가 자동으로 연동됩니다 (절대 경로 사용)
- ✅ 별도 설정이 필요 없습니다
- ✅ 관리자들에게만 `/admin/` 경로를 공유하세요

## 🔧 기술 스택

- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **API**: RESTful Table API (Cross-Origin)
- **Icons**: Font Awesome 6.4.0
- **데이터베이스**: 입주민 사이트와 공유

## 🔐 보안 권장사항

1. **별도 도메인 사용**: 입주민 사이트와 완전히 다른 도메인
2. **URL 비공개**: 관리자들에게만 공유
3. **CORS 설정**: 입주민 사이트 API가 관리자 도메인 허용 필요
4. **HTTPS 필수**: 반드시 HTTPS로 배포
5. **브라우저 인증**: 필요시 Basic Auth 또는 IP 제한 추가

## 📊 데이터 연동

관리자 사이트는 입주민 사이트의 데이터베이스에 **Cross-Origin 요청**을 보냅니다:

```javascript
// 신청 데이터 조회
GET https://입주민사이트.com/tables/pilates_contracts

// 문의 데이터 조회
GET https://입주민사이트.com/tables/pilates_inquiries

// 신청 승인/거부
PATCH https://입주민사이트.com/tables/pilates_contracts/{id}

// 문의 답변
PATCH https://입주민사이트.com/tables/pilates_inquiries/{id}
```

## 💡 사용 방법

### 레슨 신청 승인하기
1. 신청 관리 페이지 접속
2. 대기중 신청 확인
3. 상세보기 클릭
4. 서명 이미지 및 정보 검토
5. [승인] 버튼 클릭
6. → 입주민 사이트에 실시간 반영 (모집 인원 +1)

### 문의 답변하기
1. 문의 관리 페이지 접속
2. 대기중 문의 확인
3. 상세보기 클릭
4. 답변 작성
5. [답변 전송] 클릭
6. → 공개 문의는 입주민 사이트에 즉시 표시
