✅ **배포 준비 완료!**

이 프로젝트는 **하나의 도메인**에서 경로로 구분되어 작동합니다.

## 🚀 배포 방법

### 한 번에 배포
1. **Publish 탭**에서 전체 프로젝트 배포
2. 배포 완료!

### 접근 경로
```
https://your-domain.com/              → 입주민 페이지 (index.html)
https://your-domain.com/admin/        → 관리자 페이지
```

## 🔐 보안

- 입주민들은 `/admin/` 경로를 모름
- 관리자들에게만 `/admin/` URL 공유
- QR 코드는 루트(/) 경로만 포함

## ⚠️ 주의사항

- **API 경로 수정 완료**: 절대 경로(`/tables/...`) 사용으로 자동 연동
- **별도 설정 불필요**: 그대로 배포하면 됩니다
