# v12.0.11 - 📐 강사 프로필 이미지 사각형 대형화

**날짜**: 2026-03-20

## 🎯 사용자 요청

> "원형말고 사각형으로 크게 만들어줘 붉은색으로 표시한 정도의 크기가 되면 어떨까? 클릭안해도 잘보이게끔 1차적으로 만들고 그래도 안보이면 클릭해서 전체화면으로 볼 수 있는 정도로"

**문제**:
- v12.0.10에서 이미지를 200px로 키웠지만 여전히 작음
- 원형 이미지는 공간 활용이 비효율적
- 사용자가 표시한 붉은색 영역 크기 (약 500px)로 확대 필요

---

## ✅ 구현 내용

### 1. 이미지 형태 변경: 원형 → 사각형

#### 변경 전 (v12.0.10)
```css
.instructor-photo {
    width: 200px;
    height: 200px;
    border-radius: 50%;  /* 원형 */
}

.instructor-photo img {
    border-radius: 50%;  /* 원형 */
}
```

#### 변경 후 (v12.0.11)
```css
.instructor-photo {
    width: 100%;
    max-width: 500px;
    height: 500px;
    border-radius: 12px;  /* 둥근 모서리 사각형 */
}

.instructor-photo img {
    border-radius: 12px;  /* 둥근 모서리 사각형 */
}
```

**효과**:
- ✅ 원형 → 둥근 모서리 사각형 (12px radius)
- ✅ 공간 활용 효율성 증가
- ✅ 이미지 세부사항 더 많이 표시

---

### 2. 이미지 크기 대폭 확대

#### 크기 비교
| 버전 | 크기 | 증가율 |
|------|------|--------|
| v12.0.9 | 120x120px | 기준 |
| v12.0.10 | 200x200px | +67% |
| **v12.0.11** | **500x500px** | **+317%** 🚀 |

#### 상세 변경사항
```css
/* 데스크톱 */
.instructor-photo {
    width: 100%;
    max-width: 500px;
    height: 500px;
}

/* 태블릿 (768px 이하) */
@media (max-width: 768px) {
    .instructor-photo {
        height: 350px;
    }
}

/* 모바일 (480px 이하) */
@media (max-width: 480px) {
    .instructor-photo {
        height: 280px;
    }
}
```

**효과**:
- ✅ 데스크톱: 500px (사용자 요청 크기)
- ✅ 태블릿: 350px (화면 비율 유지)
- ✅ 모바일: 280px (가독성 확보)
- ✅ 클릭 없이도 세부사항 명확히 확인 가능
