## 💰 수정 #1: 회원당 가격 120,000원으로 변경

### 사용자 피드백
```
현재 회원 수가 72명이니까
120,000원 * 72 = 8,640,000원이야
```

### 문제 상황
```javascript
// ❌ 이전 코드
const avgPrice = 60000;
const monthlyRevenue = currentMembers * avgPrice;
// 72명 × 60,000원 = 4,320,000원 (실제와 다름)
```

### 해결 방법
```javascript
// ✅ 수정된 코드
const avgPrice = 120000;
const monthlyRevenue = currentMembers * avgPrice;
// 72명 × 120,000원 = 8,640,000원 ✅
```

### 매출 계산 예시

| 회원 수 | 회원당 가격 | 월 예상 매출 | 변경 전 | 변경 후 |

| 72명 | 120,000원 | 8,640,000원 | 4,320,000원 | **8,640,000원** ✅ |
| 50명 | 120,000원 | 6,000,000원 | 3,000,000원 | **6,000,000원** |
| 100명 | 120,000원 | 12,000,000원 | 6,000,000원 | **12,000,000원** |

## 🏢 수정 #2: 단지별 리포트 생성 기능

### 사용자 요구사항
```
나중에 단지수가 확장되었을때 리포트기능을 활용하려면

```

### 구현 내용

#### 1. **UI 추가: 단지 선택 드롭다운**
```html
<div class="form-group">
    <label>단지</label>
    <select id="reportComplex">
        <option value="all">📊 전체 단지 통합</option>
        <option value="cheongju-sk">청주SK뷰자이 필라테스 (cheongju-sk)</option>
        <option value="sejong-lake">세종호수공원자이 (sejong-lake)</option>
        <!-- 등록된 모든 단지 자동 표시 -->
    </select>
</div>
```

**특징**:
- "전체 단지 통합" 옵션 기본 제공
- `complex_settings` 테이블에서 자동으로 단지 목록 로드
- 단지명과 코드 함께 표시

#### 2. **단지 목록 로드 함수**
```javascript
async function loadComplexListForReport() {
    const response = await fetch('tables/complex_settings?limit=100');
    const result = await response.json();
    const complexes = result.data || [];
    
    const select = document.getElementById('reportComplex');
    select.innerHTML = '<option value="all">📊 전체 단지 통합</option>';
    
    complexes.forEach(complex => {
        const option = document.createElement('option');
        option.value = complex.complex_code;
        option.textContent = `${complex.complex_name} (${complex.complex_code})`;
        select.appendChild(option);
    });
}
```

#### 3. **데이터 필터링 로직**
```javascript
const selectedComplex = document.getElementById('reportComplex').value;

// 전체 데이터 로드
const allContracts = contractsResult.data || [];
const allCancellations = cancellationsResult.data || [];
const allPrograms = programsResult.data || [];
const allInquiries = inquiriesResult.data || [];

// 단지별 필터링 (선택된 경우)
let filteredContracts = allContracts;
let filteredCancellations = allCancellations;
let filteredPrograms = allPrograms;
let filteredInquiries = allInquiries;

if (selectedComplex !== 'all') {
    filteredContracts = allContracts.filter(c => c.complex_id === selectedComplex);
    filteredCancellations = allCancellations.filter(c => c.complex_id === selectedComplex);
    filteredPrograms = allPrograms.filter(p => p.complex_id === selectedComplex);
    filteredInquiries = allInquiries.filter(i => i.complex_id === selectedComplex);
    
    console.log(`📍 Filtered by complex: ${selectedComplex}`);
    console.log(`   Contracts: ${allContracts.length} → ${filteredContracts.length}`);
    console.log(`   Cancellations: ${allCancellations.length} → ${filteredCancellations.length}`);
    console.log(`   Programs: ${allPrograms.length} → ${filteredPrograms.length}`);
    console.log(`   Inquiries: ${allInquiries.length} → ${filteredInquiries.length}`);
}

// 필터링된 데이터로 통계 계산
const currentMembers = filteredContracts.filter(c => c.status === 'approved').length;
const monthlyRevenue = currentMembers * 120000;
```

**로직 설명**:
1. 모든 테이블에서 데이터를 가져옴 (limit=1000)
2. `selectedComplex`가 `"all"`이면 **모든 데이터 사용**
3. `selectedComplex`가 특정 단지면 **해당 단지만 필터링**
4. 필터링 전/후 데이터 수를 콘솔에 출력

#### 4. **리포트 데이터에 단지 정보 추가**
```javascript
// 단지명 가져오기
let complexName = '전체 단지';
if (selectedComplex !== 'all') {

    const selectedOption = complexSelect.options[complexSelect.selectedIndex];
    complexName = selectedOption.textContent; // 예: "청주SK뷰자이 필라테스 (cheongju-sk)"
}

currentReportData = {
    complexId: selectedComplex,
    complexName: complexName,

};
```

## 🔍 디버깅 가이드

### 단지 목록이 표시되지 않는 경우
1. **F12** → Console 확인
2. 다음 에러 확인:
   ```
   ❌ Error loading complexes for report: ...
   ```
3. Network 탭에서 `tables/complex_settings` 요청 확인
4. Status 200이면 정상, 404면 테이블 없음

### 필터링이 작동하지 않는 경우
1. **F12** → Console 확인
2. 다음 로그 확인:
   ```
   📍 Filtered by complex: cheongju-sk
      Contracts: 150 → 0  ← ⚠️ 0이면 complex_id 불일치
   ```
3. `contracts` 테이블의 `complex_id` 필드 값 확인:
   ```javascript
   fetch('tables/pilates_contracts?limit=10')
     .then(r => r.json())
     .then(d => console.log(d.data.map(c => c.complex_id)));
   ```

### 가격이 여전히 60,000원으로 표시되는 경우
1. 캐시를 강제 새로고침 (`Ctrl+Shift+R`)
2. 그래도 안되면 브라우저 재시작
3. 여전히 안되면 배포 상태 확인
