## 🎨 개선 사항

### **문제점**
- ❌ 프로그램 카드들 간의 구분이 불명확
- ❌ 시각적 계층 구조 부족
- ❌ 가독성 떨어짐
- ❌ 중요 정보 (프로그램명, 가격) 강조 부족

### **해결 방법**

#### **1. 프로그램 카드 구분 강화**
```css
/* 카드 간격 증가 */
margin-bottom: 24px

/* 테두리 강화 */
border: 2px solid #e9ecef

/* 그림자 추가 */
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08)

/* 호버 효과 */
:hover {
    border-color: #3498db
    transform: translateY(-2px)
}
```

#### **2. 헤더 영역 개선**
- 그라데이션 배경 추가
- 프로그램명 폰트 크기/두께 증가 (16px, 600)
- 아이콘 크기 증가 (20px)
- 하단 구분선 추가 (2px solid)

#### **3. 정보 그룹 시각화**
```css
/* 각 정보 그룹에 배경색 추가 */
background: #f8f9fa
border-radius: 8px
padding: 10px 14px

/* 호버 효과 */
:hover {
    background: #e9ecef
}
```

#### **4. 중요 정보 강조**
- **프로그램명**: font-weight: 600, font-size: 16px
- **월 요금**: font-weight: 600, color: #3498db (파란색)

#### **5. 간격 및 레이아웃 조정**
- info-row 간격: 14px → 16px
- info-row 하단 구분선 추가
- 패딩 증가: 20px 24px

## 🎨 시각적 개선 상세

### **1. 카드 외관**
- **테두리**: 1px → 2px (더 명확)
- **모서리**: 10px → 12px (더 부드러움)
- **간격**: 16px → 24px (더 넓음)
- **그림자**: 추가 (입체감)
- **호버**: 파란 테두리 + 살짝 들림

### **2. 헤더**
- **배경**: 그라데이션 (회색 → 흰색)
- **구분선**: 2px solid #e9ecef
- **패딩**: 18px 24px
- **아이콘**: 20px, 파란색

### **3. 정보 그룹**
- **배경**: #f8f9fa (연한 회색)
- **모서리**: 8px 둥글게
- **패딩**: 10px 14px
- **호버**: 약간 어두운 회색

### **4. 타이포그래피**
- **프로그램명**: 16px, 600 (굵게)
- **라벨**: 13px, 500 (중간)
- **값**: 14px, 600 (굵게)
- **가격**: 파란색 강조

### **5. 배지**
- **활성화**: 녹색 그라데이션
- **비활성화**: 빨간색 그라데이션
- **테두리**: 1px solid
- **패딩**: 6px 14px
