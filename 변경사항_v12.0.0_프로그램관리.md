## 🆕 신규 기능

### 1️⃣ programs 데이터베이스 테이블

**테이블명**: `programs`  
**필드 수**: 13개

| 필드명 | 타입 | 설명 |
|--------|------|------|
| `id` | text | 고유 ID |
| `complex_id` | text | 단지 ID (complex_settings 연결) |
| `program_name` | text | 프로그램명 (예: 화&목 6:1 그룹수업) |
| `program_type` | text | 프로그램 유형 (그룹/듀엣/개인) |
| `schedule_days` | text | 운영 요일 (예: 화, 목) |
| `schedule_times` | text | 운영 시간 (예: 오전 09시~11시) |
| `price` | number | 월 요금 (원) |
| `max_capacity` | number | 최대 정원 |
| `current_count` | number | 현재 신청자 수 |
| `description` | text | 프로그램 설명 |
| `is_active` | bool | 활성화 여부 |
| `display_order` | number | 표시 순서 |
| `created_at` | datetime | 생성일시 |

### 3️⃣ 프로그램 입력 폼

**입력 항목**:

#### 기본 정보
- **프로그램명** (필수): 예) 화&목 6:1 그룹수업
- **프로그램 유형** (필수): 그룹/듀엣/개인 선택
- **운영 요일** (필수): 예) 화, 목 / 월~금
- **운영 시간** (필수): 예) 오전 09시~11시, 저녁 19시~21시

#### 정원 및 요금
- **최대 정원** (필수): 숫자 입력
- **월 요금** (필수): 원 단위

#### 프로그램 설명
- **설명**: 추가 설명 (선택사항)

#### 표시 설정
- **표시 순서**: 낮은 숫자가 먼저 표시
- **활성화 여부**: 활성화/비활성화

### 4️⃣ 입주자 페이지 프로그램 동적 로드

**변경 전 (하드코딩)**:
```html
<select id="lessonType">
    <option value="">선택하세요</option>
    <option value="화&목 6:1 그룹수업">화&목 6:1 그룹수업 (오전 09시~11시...)</option>
    <option value="수&금 6:1 그룹수업">수&금 6:1 그룹수업 (오전 09시~11시...)</option>
    <!-- 고정된 옵션들 -->
</select>
```

**변경 후 (동적 로드)**:
```javascript
// 1. 데이터베이스에서 프로그램 로드
async function loadPrograms() {
    const response = await fetch('tables/programs?limit=100');
    const programs = result.data.filter(p => 
        p.is_active && 
        p.complex_id === currentComplexId
    );
    populateProgramOptions(programs);
}

// 2. 드롭다운에 동적으로 추가
function populateProgramOptions(programs) {
    programs.forEach(program => {
        const option = document.createElement('option');
        option.value = program.program_name;
        option.textContent = `${program.program_name} (${program.schedule_times}) - ${program.price}원/월`;
        lessonTypeSelect.appendChild(option);
    });
}
```

## 💡 프로그램 설정 권장사항

### 1. 초기 데이터 입력
배포 후 관리자가 다음 작업 필요:
1. admin-programs.html 접속
2. 현재 운영 중인 프로그램 모두 추가
3. 각 프로그램의 정확한 요금, 시간, 정원 입력
4. 표시 순서 설정 (인기 프로그램을 낮은 숫자로)

### 2. 프로그램 명명 규칙
**권장 형식**: `요일 + 인원 + 수업 유형`
- ✅ 좋은 예: "화&목 6:1 그룹수업"
- ✅ 좋은 예: "월수금 10:1 그룹수업"
- ❌ 나쁜 예: "그룹수업1", "프로그램A"

### 3. 프로그램 유형 구분
- **그룹**: 3명 이상
- **듀엣**: 2명
- **개인**: 1명

### 4. 정원 설정
- 현재 신청자 수는 자동으로 관리됨 (0부터 시작)
- 최대 정원만 설정하면 됨

### 5. 활성화/비활성화 활용
- 시즌 오프 프로그램: 비활성화
- 일시적으로 마감: 비활성화 (삭제하지 말 것)
- 재개 시: 다시 활성화
