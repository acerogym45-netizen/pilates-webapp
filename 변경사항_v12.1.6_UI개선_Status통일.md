## 🛠️ 수정 내역

### 1. Status 값 통일 (한글 → 영문) 🔧

#### 핵심 문제:
```javascript
// ❌ v12.1.5: 한글로 저장
status: '승인'

// 관리자 페이지 필터
filter(c => c.status === '승인')  // ✅ 카운트됨

// 정원 카운트 로직
filter(c => c.status === 'approved')  // ❌ 카운트 안 됨!
```

**결과**: 신청은 DB에 저장되지만, 정원 카운트와 관리자 페이지에 반영 안 됨

---

#### 해결책:
```javascript
// ✅ v12.1.6: 영문으로 통일
status: 'approved'

// 모든 필터링 로직
filter(c => c.status === 'approved' || c.status === '승인')  // 호환성 유지
```

**변경 파일**:
- `js/main.js` (Line 221, 321, 443, 445)
- `admin/js/admin.js` (Line 43-46)

### 4. JavaScript 함수 추가

**파일**: `js/main.js`

```javascript
// Show duplicate warning modal
function showDuplicateWarningModal(contractData) {
    const modal = document.getElementById('duplicateWarningModal');
    const content = document.getElementById('duplicateWarningContent');
    
    content.innerHTML = `
        <p><strong>동/호:</strong> ${contractData.dong}동 ${contractData.ho}호</p>
        <p><strong>성명:</strong> ${contractData.name}</p>
        <p><strong>프로그램:</strong> ${contractData.lesson_type}</p>
    `;
    
    modal.classList.add('active');
}

// Close duplicate warning modal
function closeDuplicateWarningModal() {
    const modal = document.getElementById('duplicateWarningModal');
    modal.classList.remove('active');
}

// Show success notification modal
function showSuccessNotificationModal(contractData) {
    const modal = document.getElementById('successNotificationModal');
    const content = document.getElementById('successNotificationContent');
    
    content.innerHTML = `
        <p><strong>프로그램:</strong> ${contractData.lesson_type}</p>
        <p><strong>희망 시간:</strong> ${contractData.preferred_time}</p>
        <p><strong>상태:</strong> 승인 완료</p>
    `;
    
    modal.classList.add('active');
    
    // 2초 후 기존 성공 모달도 표시 (폼 리셋용)
    setTimeout(() => {
        closeSuccessNotificationModal();
        showSuccessModal();
    }, 2000);
}

// Close success notification modal
function closeSuccessNotificationModal() {
    const modal = document.getElementById('successNotificationModal');
    modal.classList.remove('active');
}
```

### Q2. 관리자 페이지에서 승인 카운트가 0입니다
```
A: 기존 데이터가 한글 '승인'일 수 있습니다.
1. F12 → Console:
   fetch('tables/pilates_contracts?limit=10')
     .then(r => r.json())
     .then(d => console.table(d.data.map(c => ({
       name: c.name,
       status: c.status,
       lesson_type: c.lesson_type
     }))))
2. status 값 확인 ('승인' vs 'approved')
3. 둘 다 카운트되도록 수정됨 (호환성 유지)
```

### Q3. 모달이 안 닫힙니다
```
A: JavaScript 에러가 있을 수 있습니다.
1. F12 → Console에서 에러 확인
2. "closeSuccessNotificationModal is not defined" 에러 시:
   - 배포가 제대로 안 됨 → 재배포
3. 모달 배경 클릭 시에도 안 닫히는 것은 정상 (확인 버튼만)
```
