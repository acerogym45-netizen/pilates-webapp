# 📋 v8.1.0 변경사항 (2026-03-13)

## 🎯 주요 개선 사항

### 1. **프로그램 드롭다운 옵션 개선**
**변경 전**:
```
화&목 6:1 그룹수업 (09시, 10시, 11시, 19시, 20시, 21시)
수&금 6:1 그룹수업 (09시, 10시, 11시, 19시, 20시, 21시)
2:1 듀엣레슨 (13~17시 가능, 상담 필요)
1:1 개인레슨 (13~17시 가능, 상담 필요)
```

**변경 후**:
```
화&목 6:1 그룹수업 (오전 09시~11시, 저녁 19시~21시)
수&금 6:1 그룹수업 (오전 09시~11시, 저녁 19시~21시)
2:1 듀엣레슨 (오후 13시~17시, 상담 필요)
1:1 개인레슨 (오후 13시~17시, 상담 필요)
```

**개선 효과**:
- ✅ 시간대를 한눈에 파악 가능
- ✅ 오전/오후/저녁 구분 명확
- ✅ 더 자연스러운 한국어 표현

---

### 2. **개인/듀엣 레슨 입력 UI 개선**

#### 변경 전 레이아웃:
```
레이블: 희망 요일 및 시간 *
안내문: (예: 월수금 오후 2시, 화목 저녁 7시)
-----------------------------------------
[텍스트박스]
-----------------------------------------
※ 개인/듀엣 레슨은 13~17시에만 가능합니다.
※ 강사와 협의 후 최종 일정이 확정됩니다.
```

#### 변경 후 레이아웃:
```
         [텍스트박스 - 중앙 정렬]
         
희망 요일 및 시간 *
(예: 월수금 오후 2시, 화목 오후 3시)
```

**개선 사항**:
- ✅ 텍스트박스와 안내문을 중앙에 상하 배치
- ✅ 불필요한 시간 제약 안내문 삭제 (이미 드롭다운에 표시됨)
- ✅ placeholder에 명확한 예시 제공
- ✅ 깔끔하고 집중된 입력 경험

#### Placeholder 개선:
```
희망하시는 요일과 시간을 입력해주세요.

예시:
• 월수금 오후 2시
• 화목 오후 3시
• 주말 오후 4시
```

## 🚀 배포 및 테스트

### 배포 절차:
1. **Publish 탭** → **Re-deploy** 클릭
2. 배포 완료 대기 (약 30초~1분)
3. 브라우저 캐시 완전 삭제:
   - Windows: `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`
   - 또는 F12 → Application → Clear site data

### 테스트 체크리스트:

#### ✅ 입주민 페이지 테스트
- [ ] 프로그램 드롭다운에 시간대가 명확히 표시되는지 확인
  - 화&목: "오전 09시~11시, 저녁 19시~21시"
  - 개인/듀엣: "오후 13시~17시"
- [ ] 2:1 듀엣레슨 선택 시:
  - 텍스트박스가 중앙에 표시되는지
  - 안내문이 텍스트박스 아래 중앙에 표시되는지
  - placeholder 내용이 명확한지
- [ ] 1:1 개인레슨도 동일하게 작동하는지
- [ ] 그룹수업 선택 시 드롭다운으로 정상 전환되는지

#### ✅ 관리자 페이지 테스트
- [ ] 상세보기 → [수정] 클릭
- [ ] 프로그램 드롭다운 옵션이 입주민 페이지와 동일한지
- [ ] 개인/듀엣 선택 시 텍스트박스 레이아웃이 중앙 정렬되는지
- [ ] 수정 후 저장이 정상 작동하는지
