# 🔧 v9.2.1 해지 신청 수정 - 환불 계좌 제거 (2026-03-13)

## 🎯 변경 사항

### ✂️ 환불 계좌 필드 제거

**변경 이유**:
- 환불 금액은 **관리비에서 자동 차감** 처리
- 별도 계좌 이체가 필요 없음
- 입력 필드를 단순화하여 신청 편의성 향상

---

## 📋 수정된 항목

### 1️⃣ **데이터베이스 스키마 업데이트**
**테이블**: `pilates_cancellations`

**변경 전** (12개 필드):
- id, dong, ho, name, phone, lesson_type, reason, reason_detail, **refund_account** ❌, status, admin_note, created_at

**변경 후** (11개 필드):
- id, dong, ho, name, phone, lesson_type, reason, reason_detail, status, admin_note, created_at

---

### 2️⃣ **입주민 해지 신청 폼 (index.html)**

**변경 전**:
```
동/호수, 이름, 전화번호
프로그램 선택
해지 사유 선택
상세 사유 입력
환불 계좌 입력 ❌ (예: 국민은행 123-456-789012 홍길동)
```

**변경 후**:
```
동/호수, 이름, 전화번호
프로그램 선택
해지 사유 선택
상세 사유 입력
```

**안내 문구 변경**:
- 변경 전: "환불 규정에 따라 환불 금액이 결정됩니다"
- 변경 후: "환불 규정에 따라 **관리비에서 환불 금액이 차감**됩니다"

---

### 3️⃣ **관리자 해지 관리 (admin-cancellation.html)**

**상세보기 모달 변경**:
- ❌ 환불 계좌 표시 제거
- ✅ 관리자 메모 placeholder 변경: "환불 금액, 처리 내용 등을 입력하세요..."

**CSV 다운로드 변경**:
- 헤더: "환불계좌" → "상세사유"
- 데이터: `cancel.refund_account` → `cancel.reason_detail`

---
