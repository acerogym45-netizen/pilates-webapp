# 🎨 v9.1.0 UI 개선 - 공지사항 & 강사 프로필 재배치 (2026-03-13)

## 🎯 변경 사항

### 1️⃣ 공지사항 위치 변경
**변경 전**: 헤더 아래 → 입주민 페이지 중간에 표시  
**변경 후**: 헤더 바로 아래 → **최상단에 배치**

- **목적**: 중요한 공지사항을 입주민이 가장 먼저 볼 수 있도록 개선
- **표시**: 공지가 있을 때만 자동으로 표시 (is_active = true)
- **레이아웃**: 전체 너비 활용, 카테고리별 색상 구분

---

### 2️⃣ 강사 프로필 → 별도 페이지로 분리
**변경 전**: 입주민 페이지에 카드 형태로 표시  
**변경 후**: **텍스트 링크 → 별도 페이지 (instructors.html)**

#### 메인 페이지 (index.html):
- 헤더 하단에 **[강사 소개 보기]** 링크 추가
- 깔끔한 버튼 스타일 (보라색 테두리)
- hover 시 배경색 변경 + 살짝 위로 올라가는 효과

#### 강사 소개 페이지 (instructors.html):
- 독립된 페이지로 강사 정보 전체 표시
- 프로필 사진, 이름, 포지션, 경력, 소개, 자격증
- **[메인으로 돌아가기]** 버튼 제공
- 반응형 그리드 레이아웃

## 🎨 UI 흐름 개선

### Before (이전):
```
헤더
  ↓
공지사항 섹션 (중간 어딘가)
  ↓
강사 프로필 카드 (여러 개)
  ↓
신청 폼
```

### After (개선):
```
헤더
  └→ [강사 소개 보기] 링크 (클릭 → instructors.html)
  ↓
📢 공지사항 섹션 (최상단, 눈에 잘 띔)
  ↓
신청 폼
```

## 🚀 배포 및 테스트

### 재배포:
1. **Publish 탭** → **Re-deploy** 클릭
2. 완료 대기
3. 브라우저 캐시 삭제 (Ctrl+Shift+R / Cmd+Shift+R)

### 테스트 체크리스트:

#### ✅ 공지사항 (최상단 배치)
- [ ] 입주민 페이지 접속
- [ ] 헤더 바로 아래 공지사항 표시 확인
- [ ] 공지가 없으면 섹션 숨김 확인

#### ✅ 강사 프로필 링크
- [ ] 헤더 하단 **[강사 소개 보기]** 링크 확인
- [ ] 링크 클릭 → instructors.html 이동
- [ ] 강사 정보 카드 표시 확인
- [ ] **[메인으로 돌아가기]** 버튼 작동 확인

#### ✅ 샘플 데이터 추가 (테스트용)
강사가 없으면 "등록된 강사 정보가 없습니다" 메시지 표시

**강사 추가 예시** (테이블: `instructors`):
```json
{
  "name": "김필라",
  "position": "필라테스 마스터 트레이너",
  "photo_url": "",
  "bio": "10년 경력의 전문 필라테스 강사입니다. 재활 필라테스를 전문으로 합니다.",
  "experience_years": 10,
  "certifications": ["BASI Pilates 자격증", "재활 필라테스 전문가", "스포츠 마사지 자격증"],
  "display_order": 1,
  "is_active": true
}
```

**공지 추가 예시** (테이블: `notices`):
```json
{
  "title": "필라테스 수업 시간 변경 안내",
  "content": "3월 20일부터 저녁 19시 수업이 20시로 변경됩니다. 참고 부탁드립니다.",
  "category": "중요",
  "display_order": 1,
  "is_active": true
}
```

- ✅ 페이지 로딩 속도 개선
- ✅ 강사 정보 페이지에서 더 상세히 표시 가능
