## ✨ 주요 변경사항

### 1️⃣ 입주자 페이지 비밀번호 모달 중앙 정렬 수정 ✅

**문제**: 비밀번호 입력 모달이 좌측에 치우쳐서 표시됨

**해결책**:
- `index.html`의 `adminPasswordModal` 스타일 수정
- `display: flex`, `justify-content: center`, `align-items: center` 추가
- 전체 화면 오버레이 스타일 적용

**변경 내용**:
```html
<!-- Before -->
<div id="adminPasswordModal" class="modal" style="display: none;">
    <div class="modal-content" style="max-width: 400px;">

<!-- After -->
<div id="adminPasswordModal" class="modal" style="display: none; position: fixed; top: 0; left: 0; width: 100%; height: 100%; background: rgba(0,0,0,0.5); justify-content: center; align-items: center; z-index: 10000;">
    <div class="modal-content" style="max-width: 400px; width: 90%; margin: 0;">
```

**개선 사항**:
- ✅ 화면 정중앙에 모달 표시
- ✅ 반투명 배경 오버레이 추가
- ✅ 모바일 반응형 (90% 너비)
- ✅ z-index 10000으로 최상위 레이어 보장

### 2️⃣ 단지 관리 페이지 비밀번호 화면 표시 문제 수정 ✅

**문제**: `admin-complex.html` 접속 시 비밀번호 화면이 표시되지 않음

**원인 분석**:
1. sessionStorage 체크가 DOM 로드 전에 실행됨
2. `passwordScreen` 요소가 준비되기 전에 스크립트 실행
3. 비인증 상태에서 화면이 제대로 초기화되지 않음

**해결책**:
1. **DOMContentLoaded 이벤트 사용**:
   - sessionStorage 체크를 DOM 로드 후 실행
   - 비밀번호 입력 필드에 자동 포커스

2. **명시적 초기화**:
   - 비인증 시 `passwordScreen` 강제 표시 (`display: flex`)
   - `mainContent` 강제 숨김 (`display: none`)

3. **인증 상태 관리**:
   - `body.authenticated` 클래스로 인증 상태 표시
   - CSS로 인증 상태별 화면 제어

**변경된 코드**:
```javascript
// Before
if (sessionStorage.getItem('complexAuthToken') === 'authenticated') {
    showMainContent();
}

// After
document.addEventListener('DOMContentLoaded', function() {
    if (sessionStorage.getItem('complexAuthToken') === 'authenticated') {
        showMainContent();
    } else {
        // Ensure password screen is visible
        document.getElementById('passwordScreen').style.display = 'flex';
        document.getElementById('mainContent').style.display = 'none';
        
        // Focus on password input
        const input = document.getElementById('passwordInput');
        if (input) {
            input.focus();
        }
    }
});
```

**CSS 추가**:
```css
body.authenticated #passwordScreen {
    display: none;
}

body.authenticated #mainContent {
    display: block;
}
```

**개선 사항**:
- ✅ 페이지 로드 시 항상 비밀번호 화면 표시
- ✅ sessionStorage 인증 시에만 메인 콘텐츠 표시
- ✅ 비밀번호 입력 필드 자동 포커스
- ✅ Enter 키로 비밀번호 제출 가능
- ✅ 인증 상태를 body 클래스로 명확히 표시

## 💡 기술적 개선 사항

### 1. CSS 레이아웃 개선
- **Flexbox 중앙 정렬**: `justify-content: center` + `align-items: center`
- **고정 위치**: `position: fixed` + `top/left: 0`
- **전체 화면 오버레이**: `width: 100%` + `height: 100%`
- **반투명 배경**: `background: rgba(0,0,0,0.5)`

### 2. JavaScript 이벤트 처리 개선
- **DOM Ready 보장**: `DOMContentLoaded` 이벤트 사용
- **명시적 상태 관리**: `body.authenticated` 클래스
- **자동 포커스**: 사용자 경험 개선

### 3. 반응형 디자인
- **상대 너비**: `width: 90%` (모바일)
- **최대 너비**: `max-width: 400px` (데스크톱)
- **마진 제거**: `margin: 0` (중앙 정렬 보장)
