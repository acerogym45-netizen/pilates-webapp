## ✨ 새로운 기능

### 1. 🎨 UI 디자인 전면 개편 - 모던 & 미니멀

**변경 목표**:
- 기존 알록달록한 색상 → 깔끔한 회색/블루 중성 톤
- 과도한 그라데이션 제거 → 플랫 디자인
- 일관된 디자인 시스템 구축

**새로운 색상 팔레트**:
```css
--primary: #2c3e50        (다크 그레이-블루)
--accent: #3498db         (소프트 블루)
--success: #27ae60        (그린)
--warning: #f39c12        (오렌지)
--danger: #e74c3c         (레드)
--gray-50 ~ gray-800      (중성 그레이 스케일)
```

**개선된 UI 요소**:
- ✅ 입주민 페이지 (`css/style.css` - 전면 재작성 ~1,000줄)
  - 헤더, 폼, 버튼, 카드 등 모든 컴포넌트 모던화
  - 박스 쉐도우 축소, border-radius 통일
  - 폰트 크기/두께 조정
  
- ✅ 관리자 페이지 (`css/admin.css` - 전면 재작성 ~850줄)
  - 통계 카드, 필터, 테이블 디자인 개선
  - 버튼 색상 중성화
  - 일관된 레이아웃 간격

**비교 (Before vs After)**:
| 요소 | Before | After |
|-----|--------|-------|
| 주요 색상 | #667eea (보라) | #2c3e50 (회색-블루) |
| 버튼 스타일 | 그라데이션 다양 | 플랫 & 일관성 |
| 그림자 | 강한 그림자 | 미세한 그림자 |
| Border | 굵고 다양 | 얇고 통일 |
| 전체 느낌 | 활발/화려 | 전문적/깔끔 |

### 2. 🏢 다중 단지 지원 시스템 구축

**핵심 기능**:
- 여러 아파트 단지를 하나의 시스템에서 관리
- 단지별 설정 독립 관리 (프로그램, 색상, 연락처 등)
- 향후 확장 가능한 구조 설계

**새 데이터 테이블**: `complex_settings` (12개 필드)
```
- id, complex_name, complex_code
- is_active, contact_phone, contact_email
- address, logo_url, primary_color
- lesson_types (array), business_hours, display_order
```

**새 관리자 페이지**: `admin-complex.html`
- 단지 추가/수정/삭제
- 활성화/비활성화 토글
- 단지별 프로그램 설정

**확장성 확보**:
1. URL 파라미터로 단지 선택 (예: `?complex=cheongju-sk`)
2. 단지별 독립 설정 (색상, 로고, 프로그램)
3. 향후 데이터 분리 가능 (complex_id 필드 추가)

## 🎯 사용 방법

### 관리자 - 단지 관리

1. **admin-complex.html** 접속
2. **[단지 추가]** 버튼 클릭
3. 정보 입력:
   - 단지명: 청주SK뷰자이
   - 단지 코드: cheongju-sk (영문/하이픈만)
   - 연락처, 주소, 로고 URL
   - 기본 색상: #2c3e50
   - 프로그램: 쉼표로 구분
4. **[저장]** 클릭

**기존 단지 수정**:
- 카드에서 **[수정]** 버튼 클릭
- 정보 수정 후 저장

**단지 삭제**:
- **[삭제]** 버튼 → 확인
