# 🔄 변경사항 v9.2.1 - 문의사항 CSV 다운로드

## 📅 업데이트 일시: 2026-03-19

---

## ✨ 새로운 기능

### 1. 문의사항 엑셀(CSV) 추출 기능 추가

**위치**: `admin-inquiry.html` → CSV 다운로드 버튼

**기능 설명**:
- 관리자 페이지 문의 관리 탭에서 **CSV 다운로드** 버튼을 통해 엑셀로 추출 가능
- 필터링된 데이터만 다운로드 (상태 필터, 공개 필터, 검색 적용 후)
- 전체 문의 내역을 한 번에 다운로드하여 관리 용이

**CSV 파일 포함 정보**:
```
- 문의일시
- 상태 (대기/완료)
- 동
- 호수
- 이름
- 전화번호
- 공개여부 (공개/비공개)
- 제목
- 문의내용
- 답변내용
- 답변일시
```

**파일명 형식**: `필라테스문의_YYYY-MM-DD.csv`

## 🎯 사용 방법

### 관리자 - 문의 내역 다운로드

1. **admin-inquiry.html** 접속
2. (선택) 필터 적용:
   - 상태 필터: 전체 / 답변대기 / 답변완료
   - 공개 필터: 전체 / 공개 / 비공개
   - 검색: 이름, 제목으로 검색
3. **CSV 다운로드** 버튼 클릭
4. 파일 저장: `필라테스문의_2026-03-19.csv`
5. 엑셀로 열기 (한글 깨지지 않음)

## 📝 주요 개선사항

### Before (v9.2.0)
```
❌ 문의 관리 탭에 CSV 다운로드 기능 없음
❌ 문의 내역을 수작업으로 복사/정리 필요
❌ 답변 내역 백업 불가
```

### After (v9.2.1)
```
✅ CSV 다운로드 버튼 클릭 한 번으로 추출
✅ 필터링된 데이터만 선택적 다운로드
✅ 문의내용 + 답변내용 모두 포함
✅ 엑셀에서 바로 열람·정리 가능
✅ 데이터 백업 및 통계 분석 용이
```

## 🚀 배포 방법

### 1. 재배포
1. **Publish 탭** 이동
2. **Re-deploy** 버튼 클릭
3. 확인 "Yes" 클릭
4. 30~60초 대기

### 2. 테스트
1. 브라우저 캐시 삭제:
   - Windows: `Ctrl + Shift + R`
   - Mac: `Cmd + Shift + R`
   - 또는 F12 → Application → Clear site data
2. **admin-inquiry.html** 접속
3. **CSV 다운로드** 버튼 확인
4. 버튼 클릭 → 파일 다운로드 확인
5. 엑셀로 파일 열기 → 한글 정상 표시 확인
