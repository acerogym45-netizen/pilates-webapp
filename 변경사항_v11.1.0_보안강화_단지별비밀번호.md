### 1️⃣ 단지 관리 페이지 비밀번호 보호 수정 ✅
**문제**: 단지 관리 탭 클릭 시 비밀번호 화면이 표시되지 않음

**해결책**:
- `admin-complex.html`의 `showMainContent()` 함수 수정
- CSS 우선순위 문제 해결을 위해 인라인 스타일 추가
- 비밀번호: `master2026` (마스터 관리자 전용)

**변경 파일**:
```javascript
// admin-complex.html (line ~306)
function showMainContent() {
    document.getElementById('passwordScreen').style.display = 'none';
    const mainContent = document.getElementById('mainContent');
    mainContent.style.display = 'block';  // ← 추가
    mainContent.classList.add('show');
}
```

### 3️⃣ 단지별 관리자 비밀번호 시스템 구축 ✅

#### 📊 데이터베이스 스키마 업데이트
**complex_settings 테이블**에 `admin_password` 필드 추가:
```javascript
{
    name: "admin_password",
    type: "text",
    description: "입주자→관리자 페이지 접근 비밀번호"
}
```

#### 🖥️ 입주자 페이지: 5번 클릭 트리거 + 비밀번호 입력

**구현 내용**:
1. **5번 클릭 트리거** (`js/main.js`):
   - 입주자 페이지 상단 로고 텍스트를 5번 클릭 (2초 이내)
   - 비밀번호 입력 모달 표시

2. **비밀번호 입력 모달** (`index.html`):
   ```html
   <div id="adminPasswordModal" class="modal">
       <div class="modal-content">
           <h2><i class="fas fa-lock"></i> 관리자 인증</h2>
           <input type="password" id="adminPasswordInput" />
           <button onclick="checkAdminPassword()">확인</button>
       </div>
   </div>
   ```

3. **비밀번호 검증** (`js/main.js`):
   - 현재 단지의 `complex_settings`에서 `admin_password` 로드
   - 입력된 비밀번호와 비교
   - 일치 시 → `admin-main.html`로 리다이렉트
   - 불일치 시 → 오류 메시지 표시 (3초 후 자동 숨김)

**주요 함수**:
```javascript
// js/main.js
async function checkAdminPassword() {
    const complexId = complexContext.getComplexId();
    const response = await fetch(`tables/complex_settings?limit=100`);
    const result = await response.json();
    const complex = result.data.find(c => c.id === complexId);
    
    const correctPassword = complex.admin_password || 'admin1234';
    
    if (password === correctPassword) {
        window.location.href = 'admin-main.html';
    } else {
        // Show error
    }
}
```

## 🔒 보안 구조

### 2단계 보안 시스템

| 레벨 | 페이지 | 비밀번호 | 저장 위치 | 목적 |
|------|--------|----------|-----------|------|
| **Level 1** | 입주자 → 관리자 전환 | 단지별 설정 | `complex_settings.admin_password` | 각 단지 관리자가 자유롭게 설정 |

관리자 페이지 (admin-main.html)
    ↓
단지 관리 탭 클릭
    ↓
