# 🚀 초간단 배포 가이드

청주SK뷰자이 필라테스 시스템을 **3단계**로 배포하세요!

---

## ✨ 시스템 구조

```
https://your-domain.com/              → 입주민 페이지 (QR 코드)
https://your-domain.com/admin/        → 관리자 페이지 (비공개)
```

**하나의 도메인**, **하나의 배포**로 모든 게 작동합니다! 🎉

---

## 📦 1단계: 배포하기

1. **Publish 탭** 클릭
2. 전체 프로젝트 배포
3. 배포 완료! ✅

---

## 🔗 2단계: URL 확인

배포 완료 후 URL을 확인하세요.

**예시**: `https://skview-pilates.com`

### 접근 경로
- **입주민 페이지**: `https://skview-pilates.com/`
- **관리자 페이지**: `https://skview-pilates.com/admin/`

---

## 📱 3단계: QR 코드 생성

1. [QR Code Generator](https://www.qr-code-generator.com/) 접속
2. **입주민 페이지 URL** 입력 (`https://skview-pilates.com/`)
3. QR 코드 다운로드
4. 전단지와 배너에 부착

---

## 🎯 사용 방법

### 입주민
- QR 코드 스캔 → 신청 페이지
- 레슨 신청 + 문의 작성
- 공개 문의 조회

### 관리자
- `https://your-domain.com/admin/` 접속
- 신청 관리 (승인/거부)
- 문의 답변 작성
- CSV 다운로드

---

## 🔐 보안

- 입주민들은 `/admin/` 경로를 모릅니다
- QR 코드에는 루트(`/`)만 포함됩니다
- 관리자들에게만 `/admin/` URL을 공유하세요

---

## ✅ 체크리스트

배포 후 확인하세요:

- [ ] 입주민 페이지 접속 확인
- [ ] 레슨 신청 테스트
- [ ] 문의 작성 테스트
- [ ] 관리자 페이지 접속 확인 (`/admin/`)
- [ ] 신청 승인 테스트
- [ ] 모집 현황 업데이트 확인
- [ ] 문의 답변 테스트
- [ ] QR 코드 생성 완료

---

## 🆘 문제 해결

### 관리자 페이지가 안 열려요
- URL 끝에 `/admin/`을 정확히 입력했는지 확인
- 대소문자 확인 (소문자로 입력)

### 데이터가 안 보여요
- 입주민 페이지에서 신청/문의를 먼저 테스트
- 브라우저 새로고침 (F5)

### 모집 현황이 업데이트 안 돼요
- 관리자 페이지에서 **승인** 처리 확인
- 입주민 페이지 새로고침
- 대기중/거부는 카운트 안 됨

---

## 🎉 완료!

배포가 완료되면:

1. QR 코드를 전단지와 배너에 부착
2. 아파트 게시판에 게시
3. 피트니스센터에 배너 설치
4. 관리자들에게 `/admin/` URL 공유 (비공개)

**모든 준비 완료!** 🚀
