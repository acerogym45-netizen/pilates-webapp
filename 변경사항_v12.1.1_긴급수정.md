```html
<div class="contract-actions" style="display: flex; gap: 8px; flex-wrap: wrap;">
    <button onclick="openReplyModal(id)">답변하기</button>
    <button onclick="toggleHideInquiry(id, hidden)">숨김/표시</button>
    <button onclick="editInquiryContent(id)">수정</button>
```

- 수정 버튼: 회색

### 2️⃣ **비활성 프로그램 노출 기능 수정** 🔴 HIGH

#### 문제
- `display_on_inactive = true` 설정했는데도 입주자 페이지에 표시 안 됨
- 원인: `loadPrograms()` 필터링 로직이 `is_active === true`만 체크

#### 해결
필터링 로직 수정

**변경 전**:
```javascript
const programs = (result.data || [])
    .filter(p => p.is_active && p.complex_id === complexCode)
    .sort(...);
```

**변경 후**:
```javascript
const programs = (result.data || [])
    .filter(p => {
        if (p.complex_id !== complexCode) return false;
        // Show if active, or if inactive but display_on_inactive is true
        return p.is_active || p.display_on_inactive;
    })
    .sort(...);

console.log(`✅ Filtered programs: ${programs.length} (active or display_on_inactive)`);
```

#### 비활성 프로그램 표시 방식
```javascript
// Build display text
let displayText = program.program_name;
if (program.schedule_times) {
    displayText += ` (${program.schedule_times})`;
}
if (program.price) {
    displayText += ` - ${formatPrice(program.price)}원/월`;
}

// Add "곧 오픈" for inactive programs
if (!isActive) {
    displayText += ' [곧 오픈 예정]';
    option.disabled = true;      // 선택 불가능
    option.style.color = '#999';  // 회색 표시
}
```

**입주자 화면 표시**:

```
⚠️ 비활성 프로그램 (display_on_inactive=true):
  월 6:1 저녁반 (저녁 19시~21시) - 60,000원/월 [곧 오픈 예정]
```

### 3️⃣ **기존 신청자 시간대 매칭 문제 해결** 🔴 HIGH

#### 문제
- v12.1.0에서 `available_time_slots` 추가 후 기존 신청자 정원이 반영되지 않음
- 원인: 시간대 포맷 불일치
  - DB 저장 형식: "오전 09시", "저녁 19시"
  - 새 로직: "09:00", "19:00" → "오전 09시"로 변환
  - 기존 `loadTimeSlotStatus()`는 하드코딩된 시간대만 초기화

#### 해결
`loadTimeSlotStatus()` 함수 전면 개선

**핵심 변경사항**:

1. **시간 코드 매핑 추가**:
```javascript
const timeDisplayMap = {
    '09:00': '오전 09시',
    '10:00': '오전 10시',
    '11:00': '오전 11시',
    '12:00': '오후 12시',
    '13:00': '오후 13시',
    '14:00': '오후 14시',
    '15:00': '오후 15시',
    '16:00': '오후 16시',
    '17:00': '오후 17시',
    '18:00': '저녁 18시',
    '19:00': '저녁 19시',
    '20:00': '저녁 20시',
    '21:00': '저녁 21시'
};
```

2. **프로그램별 시간대 동적 초기화**:
```javascript
programs.forEach(program => {
    const availableSlots = program.available_time_slots || [];
    const timeSlotCounts = {};
    
    if (availableSlots.length > 0) {
        // Use program's available_time_slots
        availableSlots.forEach(timeCode => {
            const displayTime = timeDisplayMap[timeCode] || `${timeCode}시`;
            timeSlotCounts[displayTime] = 0;
        });
    } else {
        // Fallback: default times
        timeSlotCounts['오전 09시'] = 0;
        timeSlotCounts['오전 10시'] = 0;
        // ...
    }
    
    programTimeSlots[program.program_name] = timeSlotCounts;
    console.log(`🔧 Initialized time slots for "${program.program_name}":`, Object.keys(timeSlotCounts));
});
```

3. **시간대 정규화 개선**:
```javascript
// Old: 오전/저녁만 처리
if (time && !time.includes('오전') && !time.includes('저녁')) {
    // ...
}

// New: 오전/오후/저녁 모두 처리
if (time && !time.includes('오전') && !time.includes('오후') && !time.includes('저녁')) {
    const hour = time.replace('시', '');
    if (hour === '09' || hour === '10' || hour === '11') {
        time = `오전 ${hour}시`;
    } else if (hour === '12' || hour === '13' || hour === '14' || hour === '15' || hour === '16' || hour === '17') {
        time = `오후 ${hour}시`;
    } else if (hour === '18' || hour === '19' || hour === '20' || hour === '21') {
        time = `저녁 ${hour}시`;
    }
}
```

4. **디버깅 개선**:
```javascript
if (programTimeSlots[program] && programTimeSlots[program].hasOwnProperty(time)) {
    programTimeSlots[program][time]++;
    console.log(`✅ Counted: ${contract.name} - ${program} ${time} (Total: ${programTimeSlots[program][time]})`);
} else if (programTimeSlots[program]) {
    console.warn(`⚠️ Time slot "${time}" not found in program "${program}". Available: ${Object.keys(programTimeSlots[program]).join(', ')}`);
}
```

**결과**:
```
예시 시나리오:
프로그램: "화&목 6:1 그룹수업"
available_time_slots: ["09:00", "10:00", "12:00", "19:00"]

초기화:
{
  "오전 09시": 0,
  "오전 10시": 0,
  "오후 12시": 0,
  "저녁 19시": 0
}

기존 신청자 (DB):
- 홍길동: "오전 09시" → ✅ Counted (Total: 1)
- 김철수: "저녁 19시" → ✅ Counted (Total: 1)
- 이영희: "오전 11시" → ⚠️ Time slot not found (프로그램에 없는 시간)

최종 카운트:
{
  "오전 09시": 1,
  "오전 10시": 0,
  "오후 12시": 0,
  "저녁 19시": 1
}
```

## 🐛 문제 해결 (Troubleshooting)

### 문제 1: 비활성 프로그램이 여전히 안 보임
**증상**: `display_on_inactive = true` 설정했는데 입주자 페이지에 없음  
**원인**: 브라우저 캐시  
**해결**:
```
1. 강력 새로고침 (Ctrl+Shift+R / Cmd+Shift+R)
2. 콘솔 확인: "✅ Filtered programs: X (active or display_on_inactive)"
3. X 값이 예상과 다르면 프로그램 데이터 확인
```

### 문제 2: 기존 신청자 정원이 여전히 반영 안 됨
**증상**: 시간대에 [0/6명]으로만 표시됨  
**원인**: 프로그램명 불일치 또는 시간대 포맷 문제  
**해결**:
```
1. F12 → 콘솔 확인
2. "⚠️ Time slot not found" 경고 확인
3. 경고에 표시된 Available 시간대와 DB의 시간대 비교
4. 관리자 페이지 → 프로그램 수정 → 시간대 재설정
```

### 문제 3: 문의 관리 버튼 레이아웃 깨짐
**증상**: 버튼이 세로로 너무 길게 나열됨  
**원인**: 좁은 화면 (모바일)  
**해결**:
```
정상 동작입니다. flex-wrap: wrap으로 자동 줄바꿈됩니다.
- 데스크톱: 한 줄에 4개
- 태블릿: 2줄
- 모바일: 3-4줄
```
