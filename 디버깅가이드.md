# 🔧 프로그램별 정원 관리 디버깅 가이드

## 문제: 프로그램을 바꿔도 같은 정원이 표시됨

### 1단계: 브라우저 캐시 완전 삭제

**방법 1: 강력 새로고침**
```
Windows: Ctrl + Shift + R
Mac: Cmd + Shift + R
```

**방법 2: 캐시 수동 삭제**
```
1. F12 → Application 탭
2. 왼쪽 메뉴 "Storage" → "Clear site data"
3. [Clear site data] 버튼 클릭
4. 페이지 새로고침
```

---

### 2단계: 콘솔에서 데이터 확인

**F12 → Console 탭에서 다음을 입력**:

```javascript
// 저장된 모든 계약 확인
fetch('tables/pilates_contracts?limit=100')
  .then(r => r.json())
  .then(data => {
    console.table(data.data.map(c => ({
      이름: c.name,
      프로그램: c.lesson_type,
      시간대: c.preferred_time,
      상태: c.status
    })));
  });
```

**확인 사항**:
- `프로그램` 컬럼이 정확히 "화&목 6:1 그룹수업" 또는 "수&금 6:1 그룹수업"인가?
- `시간대` 컬럼이 "오전 09시" 형식인가?
- `상태`가 "승인"인가?

---

### 3단계: 프로그램 변경 테스트

**입주민 페이지에서**:

1. F12 콘솔 열기
2. 프로그램을 "화&목 6:1 그룹수업" 선택
3. 콘솔 확인:
   ```
   🔄 updateTimeSlotOptions called
   Selected program: 화&목 6:1 그룹수업
   Available data: { "화&목 6:1 그룹수업": {...}, "수&금 6:1 그룹수업": {...} }
   ✅ Slots for selected program: {"오전 09시": X, ...}
     오전 09시: X/6 - 모집중
     ...
   ✅ Time slots updated successfully for "화&목 6:1 그룹수업"
   ```

4. 프로그램을 "수&금 6:1 그룹수업"으로 변경
5. 콘솔 확인 (다른 숫자가 나와야 함!)

---

### 4단계: 실제 데이터 수정

**만약 기존 데이터가 잘못되었다면**:

1. 관리자 페이지 접속
2. 잘못된 신청 찾기
3. [상세보기] → [수정]
4. **프로그램 종류**를 정확히 선택:
   - "화&목 6:1 그룹수업" (& 기호!)
   - "수&금 6:1 그룹수업" (& 기호!)
5. **희망 시간대**를 정확히 선택:
   - "오전 09시" (띄어쓰기!)
6. [저장]

### 6단계: 일반적인 문제와 해결

#### 문제 1: "먼저 프로그램을 선택하세요"만 표시
**원인**: `window.programTimeSlots`가 로드되지 않음
**해결**: 페이지 완전 새로고침 (캐시 삭제)

#### 문제 2: "해당 프로그램의 데이터가 없습니다"
**원인**: 프로그램명이 정확히 일치하지 않음
**해결**: 
```javascript
// 콘솔에서 확인:
window.programTimeSlots
// 출력된 키 이름과 선택한 프로그램명이 정확히 일치하는지 확인
```

#### 문제 3: 모든 프로그램이 같은 정원 표시
**원인**: 
- 브라우저 캐시 문제
- 또는 실제 데이터가 같은 프로그램으로 저장됨

**해결**:
1. 캐시 완전 삭제
2. 재배포 후 테스트
3. 콘솔에서 실제 데이터 확인 (위 2단계)

---

### 7단계: 테스트 시나리오

#### 테스트 A: 클린 테스트
```
1. 기존 테스트 데이터 모두 삭제
2. 새 신청 2건:
   - A: 화&목 6:1 그룹수업, 오전 09시
   - B: 수&금 6:1 그룹수업, 오전 09시
3. 둘 다 승인
4. 입주민 페이지 테스트:
   - 화&목 선택 → 09시 (1/6) ✅
   - 수&금 선택 → 09시 (1/6) ✅
```

#### 테스트 B: 정원 마감 테스트
```
1. 화&목 09시 신청 6건 승인
2. 수&금 09시 신청 3건 승인
3. 입주민 페이지:
   - 화&목 선택 → 09시 (6/6) - 마감 (선택불가) ✅
   - 수&금 선택 → 09시 (3/6) - 모집중 (선택가능) ✅
```

### 긴급 대응

**지금 당장 작동하지 않는다면**:

1. **재배포**
   ```
   Publish 탭 → Re-deploy
   ```

2. **브라우저 완전 새로고침**
   ```
   Ctrl+Shift+R (또는 Cmd+Shift+R)
   ```

3. **시크릿 모드 테스트**
   ```
   Ctrl+Shift+N (Chrome)
   새 시크릿 창에서 사이트 접속
   ```

4. **콘솔 확인**
   ```
   F12 → Console
   에러 메시지 확인
   ```

5. **실제 데이터 확인**
   ```javascript
   fetch('tables/pilates_contracts?limit=100&status=승인')
     .then(r => r.json())
     .then(d => console.table(d.data))
   ```
